리스트 안에 지정된 자료형만 넣을 수 있음
자료형을 제한할 때는 기본자료형을 클래스화 한 자료형을 가져와서 사용
Wrapper Class : 기본 타입을 객체처럼 사용할 수 있도록 감싸는 클래스
기본타입               Wrapper
  int         ->      Integer
  char        ->      Character
  boolean     ->      Boolean
  float       ->      Float
  double      ->      Double
  long        ->      Long
  shor        ->      Short
  byte        ->      Byte
